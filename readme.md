## Запуск проекта

### Копирование проекта
```shell
mkdir django_stripe
cd ./django_stripe
git clone https://github.com/romich995/stripe_app.git
```

### Билдим проект
```shell
docker compose build
```


### Запуск проекта
```shell
docker compose up
```
